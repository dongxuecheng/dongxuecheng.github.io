初识jquery

Dom对象和jquery对象

声明jQuery对象时前一般加上$，Dom对象则无需
DOM对象才能使用DOM对象的方法，Jquery对象则无法使用DOM对象的方法

jquery对象与Dom对象互转

Jquery选择器

jQuery的检测机制

使用jQuery选择器获取对象时，即使对象不存在时，也不会报错。

所以检测页面上的某个元素是否存在时要使用以下代码

基本选择器

ID选择器

$(“#id”);
其中id为查询元素的ID属性值,返回一个jQuery对象，因为id是唯一的

元素选择器

$(“element”);
其中element是要获取的元素标记名，比如获取所有div元素$(“div”);

类名选择器

$(“.class”);获取所有类名为class的元素

类名选择器也有可能会返回一组jQuery包装集，因为多个元素也有可能同时为一个类名，需要使用到索引器

复合选择器

可以将多个选择器(可以是id,元素，类名选择器)组合在一起，选择器之间用,隔开

只要符合一个条件就会被匹配，返回的是集合形式的jQuery包装集，然后利用索引器获取对象

$(“.myClass,#test”),匹配所有类名为myclass和id为test的元素

通配符选择器

$(“*”),返回页面上的所有的DOM元素集合的jQuery包装集

层次选择器

ancetor descendant选择器

$(“ancetor descendant”);
ancetor可以是任何有效的选择器，descendant选择器是ancetor制定的后代元素,可以是直接子元素也可以是子孙元素
比如匹配div元素下的所有li标签$(“div li”)

parent>child选择器

使用该选择器只能选择父元素的直接子元素

prev+next选择器

$(“prev+next”);prev和next是两个相同级别的元素，且next元素要紧接着prev元素，匹配的是next元素

prev~siblings选择器

$(“prev+siblings”);prev和next是两个相同级别的元素，且siblings元素要紧接着prev元素，匹配的是所有siblings元素

过滤选择器

使用jquery操作DOM

对元素内容和值进行操作

对文本内容进行操作

通过text()方法用于获取全部匹配元素的文本内容
通过text(val)方法用于设置全部匹配元素的文本内容,通过此种方法设置后，原来的内容将被替换掉，包括html内容

对HTML内容进行操作

通过html()方法用于获取全部匹配元素的html内容，获取到第一个匹配元素的HTML内容
通过html(val)方法用于设置全部匹配元素的html内容,通过此种方法设置后，原来的内容将被替

对元素值进行操作

val() 方法返回或设置被选元素的值。元素的值是通过 value 属性设置的。如果该方法未设置参数，则返回被选元素的当前值。

对DOM节点进行操作

创建节点

插入节点

在元素内部插入

方法名 说明
append 在所选中的元素内部后追加内容
appednTo 在所选中元素加入到另一个元素内部后
prepend 在所选中的元素内部前追加内容
prependTo 在所选中元素加入到另一个元素内部前

在元素外部插入（两个元素为相同级别）

方法名 说明
after 在所选中的元素外部后追加内容
insertAfter 在所选中元素加入到另一个元素外部后
before 在所选中的元素外部前追加内容
insertBefore 在所选中元素加入到另一个元素外部前

删除节点

remove() 方法

删除被选元素及其子元素,remove() 方法也可接受一个参数，允许您对被删元素进行过滤。该参数可以是任何 jQuery 选择器的语法

empty()方法

empty() - 从被选元素中删除子元素

detach() 方法

detach() 方法移除被选元素，包括所有的文本和子节点。然后它会保留数据和事件。该方法会保留移除元素的副本，允许它们在以后被重新插入。

复制节点

clone()方法

clone() 方法生成被选元素的副本，包含子节点、文本和属性.参数类型为布尔，默认为false，不复制元素的事件处理，为true时，复制节点的事件处理

替换节点

replaceAll()方法

replaceAll() 方法把被选元素替换为新的 HTML 元素。

$(content).replaceAll(selector),content必需。规定要插入的内容（必须包含 HTML 标签）,selector必需。规定哪一个元素将被替换。被替换成content中的内容

replaceWith方法

$(selector).replaceWith(content)，selector匹配的内容被替换成content中的内容

遍历节点

each()方法

用法$(selector).each(function(index,element))，index是遍历的元素序号，从0开始，element当前的元素（也可使用 “this” 选择器），参数可选
在each方法中用$(this)表示当前所获取的对象
